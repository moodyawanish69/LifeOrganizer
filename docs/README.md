# LifeOrganizer

A modern productivity application built with React, TypeScript, and Tailwind CSS.

## Features
- Notes management
- Task tracking
- Event scheduling
- Global search
- Dark mode support

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production
```bash
npm run build
```
